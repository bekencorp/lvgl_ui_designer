# ⚙️ 属性面板

属性面板用于编辑选中组件的属性、样式和数据。

### 4.1 基础属性

#### 布局属性（Layout）

**位置和尺寸**
- **X 坐标**：组件左上角 X 轴位置（相对于父组件）
- **Y 坐标**：组件左上角 Y 轴位置（相对于父组件）
- **宽度（Width）**：组件宽度
- **高度（Height）**：组件高度

#### 组件特定属性

每种组件都有其特定的属性，例如：

**Label（标签）属性**
- **Text**：显示的文本内容
- **Long Mode**：长文本处理方式（换行/滚动/省略）


**Image（图片）属性**
- **Src**：图片源（从资源管理器选择）

**Slider（滑块）属性**
- **Min Value**：最小值
- **Max Value**：最大值

### 4.2 样式编辑器

样式编辑器是属性面板的核心功能，提供完整的 LVGL 样式支持。

#### 样式部件（Parts）

某些组件有多个样式部件，可以分别设置样式，例如：

- **Main**：主体部分
- **Scrollbar**：滚动条
- **Indicator**：指示器（如滑块的滑块部分、进度条的填充部分）
- **Knob**：旋钮（如滑块的拖动点）
- **Items**：列表项
- **Cursor**：光标

#### 样式状态（States）

LVGL 组件支持多种状态，每种状态可以有不同的样式，例如：

- **Default（默认）**：组件的默认状态
- **Checked（选中）**：复选框、开关等选中时
- **Focused（聚焦）**：组件获得焦点时
- **Pressed（按下）**：按钮按下时
- **Disabled（禁用）**：组件禁用时

#### 样式库功能

**保存样式**
1. 编辑好组件的样式
2. 点击样式面板的 "保存" 按钮
3. 输入样式名称
4. 添加描述（可选）
5. 点击保存

**应用样式**
1. 选中目标组件
2. 点击样式面板的 "应用" 按钮
3. 在样式库中选择要应用的样式
4. 点击应用

**管理样式库**
- **删除样式**：在样式库中删除
