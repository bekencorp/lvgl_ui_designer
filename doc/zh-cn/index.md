#BEKEN LVGL UI Designer 使用文档

欢迎使用 **BEKEN LVGL UI Designer**！这是一款专业的 LVGL 图形界面可视化设计工具，帮助您快速设计和开发嵌入式设备的用户界面。

![UI Designer 界面](/doc/images/workspace_design.png)


## 🎯 核心功能

### 可视化设计
- 拖拽式组件设计
- 实时预览
- 网格和参考线辅助

### 丰富的组件库
- 30+ 种 LVGL 组件
- 按钮、输入框、滑块等常用控件
- 图表、仪表盘等高级组件
- 自定义组件支持

### 强大的样式系统
- 完整的 LVGL 样式属性支持
- 多状态样式编辑
- 样式库保存和复用

### 项目管理
- 多项目管理
- 项目导入导出
- 示例模板
