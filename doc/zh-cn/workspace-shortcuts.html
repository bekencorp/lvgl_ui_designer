⌨️ 快捷键¶

掌握快捷键可以大幅提高设计效率。以下是最常用的快捷键列表。

6.1 通用操作¶

快捷键 功能 说明
Ctrl + S 保存项目 手动保存当前项目
Ctrl + Z 撤销 撤销上一步操作
Ctrl + Shift + Z 重做 重做上一步撤销的操作
Ctrl + Y 重做 重做（与 Ctrl+Shift+Z 相同）

6.2 视图操作¶

快捷键 功能 说明
滚动滚轮 缩放画布 滚动鼠标滚轮缩放画布（50% - 200%）
Ctrl + 滚轮 固定左上角缩放 保持画布左上角不动进行缩放
Delete 删除参考线 选中参考线后按 Delete 键删除

6.3 组件操作¶

快捷键 功能 说明
Ctrl + C 复制组件 复制选中的组件
Ctrl + V 粘贴组件 粘贴已复制的组件
Delete 删除组件 删除选中的组件

6.4 组件调整¶

快捷键 功能 说明
拖拽边角 调整大小 拖拽组件边角控制点调整大小
Shift + 拖拽 等比缩放 按住 Shift 拖拽保持宽高比
