# 工作台使用指南

工作台是BEKEN LVGL UI Designer 的核心工作区域，本指南详细介绍工作台各个模块的功能和使用方法。

## 📐 工作台布局

工作台主要分为 5 个功能区域：

![UI Designer 界面](/doc/images/workspace_empty.png)

### 五大功能区域

1. **[画布区域](workspace-canvas.md)** - 可视化设计的核心区域
2. **[组件库](workspace-components.md)** - 丰富的 UI 组件库
3. **[组件树](workspace-tree.md)** - 组件层级结构管理
4. **[属性面板](workspace-properties.md)** - 属性和样式编辑
5. **[工具栏](workspace-toolbar.md)** - 项目管理和常用功能

### 快捷键

查看完整的快捷键列表，请参考 **[快捷键](workspace-shortcuts.md)** 文档。

## 📚 详细文档

点击下方链接查看各模块的详细使用说明：

### 核心功能模块

- **[🎨 画布区域](workspace-canvas.md)**
  - 视图控制（缩放、平移、网格）
  - 标尺和参考线
  - 组件操作（添加、选择、移动、调整大小）
  - 组件对齐、层级、锁定和隐藏

- **[🧩 组件库](workspace-components.md)**
  - 系统组件库（30+ 种组件）
  - 组件分类和搜索
  - 自定义组件库的创建和使用

- **[🌲 组件树](workspace-tree.md)**
  - 页面管理（创建、切换、删除、重命名）
  - 组件层级显示
  - 组件操作（选择、重命名、拖拽排序）

- **[⚙️ 属性面板](workspace-properties.md)**
  - 基础属性编辑
  - 样式编辑器（部件、状态）
  - 样式库功能

- **[🛠️ 工具栏](workspace-toolbar.md)**
  - 项目信息和保存状态
  - 编辑操作（撤销/重做）
  - 视图控制
  - 资源管理器
  - 预览及代码生成

### 其他

- **[⌨️ 快捷键](workspace-shortcuts.md)**
  - 通用操作快捷键
  - 组件操作快捷键
  - 组件调整快捷键
