# ⌨️ Keyboard Shortcuts

Mastering keyboard shortcuts can greatly improve design efficiency. Below is a list of the most commonly used shortcuts.

### 6.1 General Operations

| Shortcut | Function | Description |
|----------|----------|-------------|
| `Ctrl + S` | Save Project | Manually save current project |
| `Ctrl + Z` | Undo | Undo previous operation |
| `Ctrl + Shift + Z` | Redo | Redo previously undone operation |
| `Ctrl + Y` | Redo | Redo (same as Ctrl+Shift+Z) |


### 6.2 View Operations

| Shortcut | Function | Description |
|----------|----------|-------------|
| `Scroll Wheel` | Zoom Canvas | Scroll mouse wheel to zoom canvas (50% - 200%) |
| `Ctrl + Wheel` | Fixed Top-Left Zoom | Zoom while keeping canvas top-left corner fixed |
| `Delete` | Delete Guideline | Select guideline and press Delete key to delete |

### 6.3 Component Operations

| Shortcut | Function | Description |
|----------|----------|-------------|
| `Ctrl + C` | Copy Component | Copy selected component |
| `Ctrl + V` | Paste Component | Paste copied component |
| `Delete` | Delete Component | Delete selected component |

### 6.4 Component Adjustment

| Shortcut | Function | Description |
|----------|----------|-------------|
| Drag Corners | Resize | Drag component corner control points to resize |
| `Shift + Drag` | Proportional Scaling | Hold Shift while dragging to maintain aspect ratio |
